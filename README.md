Доброго времени суток!

Проект выполнил: Уразов Станислав Олегович.

Для запуска проекта необходимы: файл Urazov_car_proj.exe и папка rsc.
Необходимо, чтобы файл и папка находились в одной папке, на одинаковом уровне видимости.

Выполнено:

Необычная форма движения (по спирали);
Регулирование параметров движения: скорость, время появления машин и ускорение торможения.
Новые параметры будут использоваться для генерации новых машин на дороге,
они не затронут уже созданные машины;
Кнопка остановки и возобновления движения (поломки и починки) одной машины;
Кнопка пересоздания процесса;
Кнопка остановки и возобновления всего процесса движения машин;
Состояния движения машин:
Зелёный - обычное движение,
Оранжевый - торможение,
Красный - авария двух машин (возобновляют движение после небольшого перерыва),
Мигающий - машина сломана.


Также реализованы:
Графическая составляющая приложения;
Изменение цвета машин.
